# Benötigte Bilder — Laufladen Kassel

Alle Bilder als JPG (progressiv), optimiert für Web. Dateien hier ablegen, dann in den jeweiligen Komponenten einbinden.

## Pflichtbilder

| Dateiname | Maße | Verwendung | Hinweis |
|---|---|---|---|
| `hero-laufladen.jpg` | 1600 × 900 px | Hero-Section Startseite | Schaufenster oder Laden-Innenansicht, hell & einladend |
| `og-image.jpg` | 1200 × 630 px | Open Graph / Social Sharing | Logo + Adresse auf hellem Hintergrund |
| `laufanalyse-hero.jpg` | 1600 × 900 px | Hero Laufanalyse-Seite | Laufband-Aufnahme oder Beratungssituation |
| `beratung-hero.jpg` | 1600 × 900 px | Hero Beratungs-Seite | Schuhberatung im Laden |

## Technische Anforderungen

- Format: **JPG** (progressiv), alternativ **WebP** mit JPG-Fallback
- Farbprofil: sRGB
- Qualität: 80–85 % (Kompromiss Qualität / Dateigröße)
- Max. Dateigröße: Hero ≤ 300 KB · Team ≤ 80 KB · OG ≤ 150 KB
- Empfohlenes Tool: [Squoosh](https://squoosh.app) oder `sharp` CLI
